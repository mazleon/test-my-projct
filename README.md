# Continious Integration and Continious Delivery - Learning with Hands-on

# Overview
This project is a simple web application that is used to demonstrate the concepts of Continious Integration and Continious Delivery. 

# Tech Stack 
- Python
- FastAPI
- Docker
- Github Actions

# Workflow
- Push to main branch
- Github Actions will run the tests
- If tests pass, Github Actions will build the Docker image
- If Docker image is built successfully, Github Actions will deploy the application to AWS EC2

# How to run the application
- Clone the repository
- Install the dependencies
- Run the application

# How to test the application
- Run the tests
- Check the logs
- Check the application
